### Load Testing

#### Basic Load Testing
```bash
# Install artillery for load testing
npm install -g artillery

# Create basic load test
echo '
config:
  target: "https://themoonexports.firebase.app"
  phases:
    - duration: 60
      arrivalRate: 10
scenarios:
  - name: "Homepage visit"
    requests:
      - get:
          url: "/"
' > load-test.yml

# Run load test
artillery run load-test.yml
```
